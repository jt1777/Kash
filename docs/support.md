## FAQs

Browse the **#faqs** channel for answers to common questions about deposits, redemptions, batch timing, fees, and wallet setup. Check there first — many issues are already addressed.

---

## Technical support

If something is broken, stuck, or not working as expected — a failed transaction, missing claim, wrong network, or app bug — open a ticket in the **#support** channel.

When posting, include:

- Wallet address (optional but helpful)
- Product tab (KASH-ETH or KASH-BTC)
- Transaction hash, if you have one
- A short description of what you tried and what happened

The team uses **#support** for technical issues that need follow-up. General how-to questions belong in **#faqs** when possible.
