## Technical Moat

### Why Agents Choose KASH

1. **Lower cognitive overhead**
   - No need to monitor positions
   - No liquidation risk (delta neutral)
   - No rebalancing needed

2. **Composable**
   - ERC20 KashEth can be used in other protocols
   - Use as collateral
   - Transfer between agents

3. **Gas Efficient**
   - Batched deposits (not per-transaction)
   - Optimized for Arbitrum L2
   - Minimal state updates

4. **Trustless**
   - No admin keys
   - Immutable logic
   - Transparent on-chain
